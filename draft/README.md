## なぜこの段階を踏むのか

「サイト自体が作品」という方向性を選んだ場合、デザイン → 実装 と進む前に
**何を語るサイトなのか** を先に固めないと、装飾だけ豪華で中身の薄い
ポートフォリオになりがちです。実績のある人ほど起こりやすい罠なので、
コピー先行で進めることを推奨しています。
