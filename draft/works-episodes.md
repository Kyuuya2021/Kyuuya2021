# Works Episodes — 作品ごとの語り台本

> 各プロジェクトに「きっかけ / やっていること / いま面白いところ」の3点を埋める。
> README に列挙された情報を、エピソードに**翻訳**するための作業ファイル。
> ここに書いた本人の言葉が、後で Works セクションのカード本文になります。

---

## テンプレート

```
### [プロジェクト名]
- 軸: Art / Tech / Entrepreneurship / Research のどれか
- 期間: 開始 – 現在 / アーカイブ
- ステータス: live / wip / archive

[1] きっかけ (Why) — 1 行
    なぜ始めた？ 誰に呼ばれた？ どこで火がついた？

[2] やっていること (What) — 2-3 行
    具体的に。固有名詞・数字・場所を入れる。

[3] いま何が面白いか (Now) — 1 行
    最近気づいたこと、視点が変わった瞬間。

[4] メディア
    - 写真 (何の写真があれば理想か)
    - 動画 (もしあれば)
    - 外部リンク
```
