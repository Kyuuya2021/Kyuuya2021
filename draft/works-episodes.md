# Works Episodes — 作品ごとの語り台本 (v2)

> 各プロジェクトに「きっかけ / やっていること / いま面白いところ / otium との接続」の
> 4 点を埋める。各 Works に **otium との接続** を必ず 1 行入れることで、
> 思想の一貫性が "見える" 構造になる (これが第2版の最大の変更)。
>
> 優先順は portfolio-brief.md 第8節と同期。

---

## テンプレート

```
### [プロジェクト名]
- 軸: AI/個人開発 / プライア / 学業・研究 / コーヒー&カフェ / アート&画商 / 習慣
- 期間: 開始 – 現在 / アーカイブ
- ステータス: live / wip / archive

[1] きっかけ (Why) — 1 行
    なぜ始めた？ 誰に呼ばれた？ どこで火がついた？

[2] やっていること (What) — 2-3 行
    具体的に。固有名詞・数字・場所を入れる。

[3] いま何が面白いか (Now) — 1 行
    最近気づいたこと、視点が変わった瞬間。

[4] otium との接続 — 1 行 ★ 必須
    この活動が、人々の "暇" を作る / 自分の "暇" を実践することに、
    どう繋がっているか。

[5] メディア
    - 写真 (何の写真があれば理想か)
    - 動画 (もしあれば)
    - 外部リンク
```
