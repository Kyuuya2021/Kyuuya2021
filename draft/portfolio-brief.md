## 0. 中核思想（最重要）

> **「忙しさは、生きていることではない。
>   生きるための "暇" を、世の中に増やしに行っている。」**
> ― Yuya Kawasome

思想的基盤: **セネカ『人生の短さについて』(De Brevitate Vitae, 49 AD) / ストア哲学**

| ラテン語 | 概念 | 現代訳 | 川染の仕事との対応 |
|---|---|---|---|
| **otium** | 自由な時間／観想・鍛錬のための時間 | 「真の暇」 | これを **作りに行っている** |
| **negotium** | 業務／占有された状態 | 「忙しさ」 | これに人々は **奪われている** |
| **occupati** | 占有されている人々 | 「自分の人生を生きていない人」 | 解放したい対象 |

中核引用:

> *Vita, si uti scias, longa est.*
> 「人生は、使い方さえ分かっていれば、長い。」
> ― Seneca, *De Brevitate Vitae*, I, 4

> *Non accipimus brevem vitam sed fecimus.*
> 「我々は短い人生を授かったのではない。我々がそれを短くしているのだ。」
> ― 同 I, 4

セネカの言う *otium* は **怠惰ではない**。哲学・読書・自己鍛錬・対話のための時間。
川染さんの活動 (AI / 画商 / カフェ / Stanford / ランニング / 読書) はすべて、
**人々の otium を増やす** か、**自分の otium を実践する** ものとして繋がる。

→ サイト全体がこの思想で貫かれる。装飾ではなく、構造として。

## 5. Voice プロファイル（Seneca / Stoic 軸）

**3語**: *静かな強度 / 反・忙しさ / 古典と現場*

サイト全体の語り口は、**Seneca が現代の起業家として書いたら、こうなる** と
仮定すると整いやすい。哲学的だが説教臭くない、私的だが普遍的、短いが余韻がある。

### Do
- 動詞で書く（「作っている」「いる」「読んでいる」）
- 固有名詞を出す（「ルワンダの」「d.school の廊下で」「セネカが書いたように」）
- 一人称で書く（「自分は」/ "I am")
- 1段落 3 文以内、句点で止める短文を混ぜる
- "暇" "生きる" "充実" "時間" の語彙を避けず、堂々と使う
- 必要な場面では古典 (セネカ/エピクテトス/マルクス・アウレリウス等) を引く

### Don't
- 「最先端」「次世代」「イノベーション」「DX」「効率化」「時短」(これらが特に NG ―― 中核思想と逆行)
- 受賞リストを文中に詰め込む (年表が別にある)
- 「〜させていただいております」（距離を取りすぎる）
- すべてを完成形で語る（伸びしろが消える）
- 「忙しい」を肯定的に使う（思想と矛盾する）

### サンプル文 (Before / After)
- 🔴 NG: 「最先端の AI 技術を駆使し、革新的なソリューションを提供しています。」
- 🟢 OK: 「ルワンダのコーヒー農家のために、AI で農地を見張る仕組みを作っている。3 年前、JICA の人に呼ばれて現地に飛んだのが始まりだった。」

- 🔴 NG: 「効率化により、お客様の時間を節約します。」
- 🟢 OK: 「節約した時間で何をするか、の方が、節約そのものより難しい。だから自分は、節約した先の "時間の使い方" まで設計したい。」

## 6. Hero コピー（確定）

```
忙しさは、生きていることではない。
生きるための "暇" を、世の中に増やしに行っている。
                                  — Yuya Kawasome
```

英訳 (バイリンガル切替時、意訳寄り):
```
Being busy is not the same as being alive.
I'm going out to multiply the kind of "leisure"
that lets people actually live.
                              — Yuya Kawasome
```

> 採用根拠: 効率化を売るスタートアップ業界への明確な逆張り。Seneca の otium /
> negotium の対比を、現代日本語で言い直したもの。短く、強く、誤読しにくい。

### Hero 周りの演出仮案

- 1 行目「忙しさは、生きていることではない。」だけ最初に出す。3 秒スティル。
- 2 行目がフェードイン。
- 背景: 川染さんの「動いている時間」(走っている / 読んでいる / コードを書いている / 画廊にいる) を超低速のシネマグラフで。
- スクロール促進の指示は出さない。沈黙を作る。

### 旧候補 (A〜E) のアーカイブ

第1版で検討した A〜E は、以下の理由で見送り:
- A (矛盾並列), C (場所と時間) ← 本人より却下
- B (問い先行) ← Seneca 軸が定まり、より弱い表現になったため降格
- D (革命家ストレート) ← サブコピー / About の中で活用余地あり
- E (動詞だけ) ← 別セクション (例: Works 導入) で再利用余地あり

## 9. Library / 本棚セクション (確定)

サイトの中核セクションのひとつ。**「いま自分を作っているもの」を見せる場所**。
読書 / YouTube / Podcast を一緒に並べることで、思考の食事内容を開示する。

### 構造

```
📖 BOOKS
  ├─ 古典      (Classics)      — Seneca, Marcus Aurelius, ...
  ├─ 実用書    (Practical)
  ├─ 小説      (Fiction)
  └─ 漫画      (Manga)

🎧 LISTENING
  ├─ YouTube   (好きなチャンネル)
  └─ Podcast   (好きな番組)

各タブの中で:
  ▸ Now            (今読んでる/聴いてる)  — 1〜2件
  ▸ Influences     (自分を作った)         — 3〜5件
  ▸ Reread / Re-listen (何度も)          — 3〜5件
```

各アイテムは `タイトル / 著者 or チャンネル名 / "なぜ好きか" 1行` の3点セット。

### このセクションの戦略的役割

1. **Voice の証拠** ―― 「セネカが好き」と言うより、棚にあった方が説得力 10 倍
2. **「面白い人」を出す装置** ―― 人柄は本棚に出る、漫画があると一気に距離が縮まる
3. **更新で寿命が伸びる** ―― 新しい本を追加するだけで「動いている」感が出る
4. **思想の証明** ―― セネカが古典の 1 冊目に置かれていることで、Hero の言葉が裏付けられる

### インタラクション (実装フェーズで検討)

- 背表紙が物理的に並んでいて、クリックで「抜き出される」アニメーション
- 抜き出した本の見開きに引用 + 注釈
- 「閉じる」と棚に戻る音が鳴る (オプション)

データの書き溜めは `draft/bookshelf.md` で進める。

## 9.5 Design Language 確定シグナル (v2)

> Library の手塚治虫まとめから派生した、サイト全体のデザイン方向。
> Phase 2 (ChatGPT 壁打ち) と Phase 3 (Claude design) で常に参照する。

### 中核キーワード

| 採用 | 不採用 |
|---|---|
| 勢い (線とタイポで出す、色ではなく) | 静謐さ単独 |
| 重さ (古典的な重み、産業的ではなく) | 軽さ単独 |
| 人間らしさ | 無菌・整然 |
| 現実味 | ファンタジー単独 |
| 手の痕跡 | 完全デジタル |
| 厚かましさ (良い意味で) | 上品すぎる |
| 編集デザイン的な強い人格 | ポップ・かわいい・キャッチー |
| 淡い色 (ベージュ・グレー・白黒) ★ | 鮮やかな色面 |

### 視覚モチーフ (確定)

採用:
- **墨**: 滲む、垂れる、勢いのある一筆
- **古文書**: 余白、明朝、文字の重さ
- **手塚治虫的なペン入れ**: 線が物語を運ぶ強さ
- **朝の光**: 神聖さではなく、生活の中の強い光

不採用 (本人より):
- ❌ 木版画 / 浮世絵 (= 不要)

### タイポグラフィ (確定方向)

本人の好みに基づき、以下の **3 つの混在** を狙う:

- **太め明朝** (a): 游明朝 Demibold / A1明朝 — 和文の主役
- **古典セリフ欧文** (c): GT Sectra / Italian Old Style — 欧文・引用・Latin
- **手書き的書体** (e): ペン入れ感のある日英 — 装飾・サイン・余白の点描

避ける: 重いゴシック (b)、黒い無骨セリフ (d)。
→ 「**重さは出すが、無骨ではない**」 ―― これが川染さんのタイポの輪郭。

### モーション初期方向

- イージング: 物理を感じる (重力 / 質量 / 摩擦)
- 出現: 墨が滲む、線が引かれる、紙がめくれる
- 消失: ふっと消えるより、動きの余韻を残す
- 避ける: 軽快なフェード、機械的なスライド、跳ねる動き

### カラー方向 (確定 / 淡め基調)

「**和紙の上に、墨と微かな彩り**」がコンセプト。

```
墨 (#1c1a17 〜 #2c2925)         ← 主要文字 / 線、温かいブラック
和紙 (#f5f1e8 〜 #ebe5d6)       ← 背景の温度、生成りベージュ
鼠色 (#7c7977)                   ← 補助文字、罫線、薄墨
利休白茶 (#cdc6b8)               ← セクション境界の和紙
ほぼ白 (#fbfaf6)                ← カード背景

[アクセント (使用は最小限、必要な箇所のみ)]
薄朱 (#b94a3a → 透明度を上げて使う) ← 引用、強調点
淡群青 (#3a4a6b → 透明度を上げて使う) ← セネカ・古典領域
```

→ 鮮やかな色面ではなく、**紙の上に墨が落ちる温度** で構築する。
→ ChatGPT 画像生成では「ベージュ・グレー・白黒の淡いトーン」を最初の指示に入れる。

### 参考にした (が、そのまま真似はしない) サイト

- **fumino.b-rave.tokyo** ― 編集デザイン的な強い人格は欲しい
  ただし「**ポップな方向は違う**」(本人言及)
  抽出する要素: 強いキャラクター / 手の感じ / 余白の使い方 / 文字の存在感
  抽出しない要素: 鮮やかな色 / 親しみやすさ / 軽快さ

【サイトの中核思想】
- セネカ『人生の短さについて』(De Brevitate Vitae) を北極星にしている
- Hero コピー: "忙しさは、生きていることではない。
                生きるための『暇』(otium) を世の中に増やしに行っている。"
- 効率化を売るスタートアップ業界への明確な逆張り
- 怠惰ではなく充実 / 短い人生を、長く生きるために

【視覚的な美学 ★ 最重要】
- 「勢いと重さ、人間らしさと現実味」が中核
- 「重さ」は色ではなく、線とタイポで出す (色は淡い)
- 手塚治虫が好き ―― 線そのものが力を持ち、物語を運ぶ
- "きれいに整える" 方向ではなく、"勢いと重さで押し切る" 方向
- ただし産業的・無骨ではない。古典的な重み。

【カラー方向 ★ 重要】
- 基調: 淡い色 / ベージュ / グレー / 白黒
- 「和紙の上に、墨と微かな彩り」というコンセプト
- 鮮やかな色面は不要。紙の温度と墨の濃度で勝負
- アクセントを使う場合も、薄朱・淡群青などくすんだ色を最小限

【タイポグラフィ方向】
- 和文: 太めの明朝 (游明朝 Demibold / A1明朝 のような重さ)
- 欧文: 古典セリフ (GT Sectra / Italian Old Style 系)
- 装飾: 「実際に手で書いた」ような書体を点描的に混ぜる
- 重いゴシックや黒い無骨セリフは避ける

【視覚モチーフ (採用)】
- 墨: 滲む、垂れる、勢いのある一筆
- 古文書: 余白、明朝、文字の重さ、紙の質感
- 手塚治虫的なペン入れ: 線が物語を運ぶ強さ
- 朝の光: 神聖さではなく、生活の中の強い光

【視覚モチーフ (不採用)】
- 木版画 / 浮世絵 (本人より明示的に外す)

【サイトのトーン】
- 実験的でインタラクティブ。「サイトそのものが作品」
- 古典 (ストア哲学) と現代テクノロジーの交差点
- 静かな強度。説教臭くない、私的だが普遍的、短いが余韻がある
- 矛盾を恐れない (画商とドローン、古典と AI を同居させる)
- 余白が多い、編集デザイン的、強い人格
- 手の痕跡が残る (整いすぎない、デジタルすぎない)

【参考 (抽出するもの / しないもの)】
- 参考にしたサイト: fumino.b-rave.tokyo
- 抽出する: 編集デザイン的な強い人格 / 手の感じ / 余白の使い方 / 文字の存在感
- 抽出しない: ポップさ / 鮮やかな色 / 親しみやすさ / 軽快さ

【避けたいもの】
- ありがちな「テックスタートアップ」感 (ネオン、サイバーパンク、グリッド)
- 「効率化」「時短」「最先端」を匂わせるビジュアル (思想と逆行)
- 整いすぎた写真、ストックフォト感、無菌な美意識
- 自己啓発書の表紙のような押し付けがましさ
- 細い線、軽すぎる Apple マーケティング系の美学
- ポップ、かわいい、キャッチー、illustration heavy

【欲しいもの (5案ほど)】
1. キーヴィジュアル候補 (Hero に置く / 動きあり推奨)
   ―― 淡い紙の上に、墨と勢いのある線、太め明朝のコピー
2. カラーパレット (5色 / すべて淡め、ベージュ・グレー・白黒中心)
3. タイポグラフィの組み合わせ案 (和文太め明朝 × 欧文古典セリフ × 手書き)
