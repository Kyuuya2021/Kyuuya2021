## 使い方メモ

- **ChatGPT / Claude artifacts** にそのまま貼り、出力された `index.html`
  をローカルで開いて検証する。
- 出力が薄ければ「もっと余白を / 太め明朝を強く / 墨の滲みを SVG で」など
  ピンポイントで追記指示する。
- 気に入った版が出たら Phase 4 (Cursor + Claude Code CLI) で Next.js に
  移植する。このプロトタイプは「方向の合意」を取るためのもの。
- React 版が欲しい場合は § 7 の「React 不可」を「単一 React コンポーネント
  (Tailwind 不可、styled-jsx か CSS-in-JS で)」に差し替える。

## 出力後にチェックすること

1. Hero コピーが一字一句正しいか
2. About 末尾と Footer にセネカ引用があるか
3. Works タイルに "otium との接続" の行が入っているか
4. 主背景が #f5f1e8 (和紙色) になっているか
5. 禁止語 (最先端/効率化/etc) が混入していないか
6. スクロール促進矢印が出ていないか
